Q: What is a React island?
A: A small interactive React component mounted into an otherwise static HTML page, instead of taking over the whole site.

Q: Why bundle React for a Jekyll site rather than rewriting it?
A: You keep all existing posts, layouts, and SEO untouched and only pay the cost of React where you actually need interactivity.

Q: What does motion/react replace?
A: It's the successor to framer-motion — same API surface, framework-agnostic core, imported from `motion/react` for React projects.
